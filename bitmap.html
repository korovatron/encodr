Bitmapped Images

Try to recreate a gallery image by entering denary values (0-255) into each byte field. In 1-bit mode each byte controls 8 pixels, and in 2-bit mode each byte controls 4 double-width pixels. Three bytes fill one row of 24 pixels — the same format as a Commodore 64 sprite.
